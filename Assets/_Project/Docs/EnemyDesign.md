## 👾 적 시스템 개요

### 기본 구조

```
모든 적의 공통 요소:
├─ 스탯
│  ├─ HP (체력)
│  ├─ 공격력 (기본 공격 데미지)
│  └─ 방어력 (선택적)
│
├─ 행동 시스템
│  ├─ 행동 패턴 리스트
│  ├─ 행동 선택 확률
│  └─ 다음 행동 예고
│
└─ 특수 능력 (선택적)
   ├─ 조건부 발동
   └─ 페이즈 전환
```

### 적 분류

```
난이도별:
├─ 초급 (Tutorial): 슬라임
├─ 중급 (Normal): 고블린
└─ 보스 (Boss): 오크

역할별 (Post-MVP):
├─ 공격형: 높은 데미지, 낮은 HP
├─ 방어형: 낮은 데미지, 높은 HP + 방어
├─ 보조형: 버프/디버프 중심
└─ 특수형: 소환, 변신 등
```

## ⚔️ 행동 시스템

### 행동 종류

#### 1. 공격 (Attack)
```
기본 공격:
- 단순 데미지
- 플레이어 방어력 → HP 순서로 감소
- 가장 기본적인 행동

예시:
"슬라임이 몸을 부풀려 공격한다!"
→ 플레이어에게 5 데미지
```

#### 2. 강공격 (Heavy Attack)
```
고위력 공격:
- 기본 공격의 1.5-2배 데미지
- 1턴 전에 예고
- 플레이어가 대비할 시간 제공

예고 시스템:
턴 N: "고블린이 큰 칼을 휘두를 준비를 한다!" (예고)
턴 N+1: 강공격 실행 (12 데미지)

전략적 가치:
- 플레이어에게 방어 준비 시간
- 예측 가능한 위협
- 리스크-보상 균형
```

#### 3. 방어 (Defend)
```
방어력 획득:
- 적이 일시적으로 방어력 획득
- 다음 공격 데미지 감소
- 플레이어의 강공격 유도

메커니즘:
적 방어력 10 → 플레이어 공격 25 → 실제 데미지 15

전략적 의미:
- 플레이어의 약한 공격 비효율
- 강공격 모으기 유도
- 전투 템포 조절
```

#### 4. 버프 (Buff)
```
자기 강화:
- 공격력 증가 (N턴)
- 방어력 증가 (N턴)
- 회피율 증가 (N턴)

예시:
"오크가 분노하며 힘이 솟아오른다!"
→ 공격력 +5 (3턴 지속)

적용:
- 공격력 15 → 20 (버프 중)
- 버프 종료 시 원래대로 복귀
```

#### 5. 디버프 (Debuff)
```
플레이어 약화:
- 방어력 감소
- DOT (지속 데미지)
- 공격력 감소

예시:
"고블린이 독칼로 베었다!"
→ 플레이어에게 독 부여 (2 데미지/턴, 3턴)

전략적 대응:
- 빠른 처치 필요
- 회복/정화 필요
```

### 행동 선택 시스템

```
기본 메커니즘:
1. 행동 패턴 리스트에서 가중치 기반 랜덤 선택
2. 다음 행동 예고 (플레이어에게 표시)
3. 다음 턴에 행동 실행

예시 (고블린):
행동 패턴:
- 공격 (60%): 데미지 8
- 강공격 (40%): 데미지 12 (1턴 예고)

실행 순서:
턴 1: 랜덤 선택 → 강공격 → "다음: 강공격!" 예고
턴 2: 강공격 실행 → 랜덤 선택 → "다음: 공격!" 예고
턴 3: 공격 실행 → ...
```

### 예고 시스템

```
UI 표시:
┌─────────────────────────┐
│   [고블린 그림]          │
│   HP: ████░░ 35/50      │
│                         │
│   다음 행동:            │
│   💥 강공격 (12)        │  ← 경고
└─────────────────────────┘

시각적 피드백:
- 강공격: 빨간 오라 + ⚠️ 아이콘
- 방어: 파란 쉴드 + 🛡️ 아이콘
- 버프: 노란 빛 + ✨ 아이콘

목적:
✅ 플레이어에게 정보 제공
✅ 전략적 대응 가능
✅ 공정한 난이도
✅ 긴장감 조성
```

## 🎯 MVP 적 3종

### 1. 슬라임 (초급 - Tutorial Enemy)

#### 기본 정보
```
이름: 슬라임
역할: 튜토리얼 적
난이도: ⭐☆☆☆☆
출현: 1-3번째 전투
```

#### 스탯
```
HP: 30
공격력: 5
방어력: 0
특수능력: 없음
```

#### 행동 패턴
```
1. 공격 (80%)
   데미지: 5
   "슬라임이 몸을 부풀려 덤벼든다!"

2. 방어 (20%)
   방어력: +3 (1턴)
   "슬라임이 몸을 단단하게 만든다!"
```

#### 밸런싱
```
목표 전투 시간: 3-5턴
예상 턴 진행:

턴 1 (플레이어):
  공격 15 → 슬라임 HP 30 → 15

턴 1 (슬라임):
  공격 5 → 플레이어 HP 100 → 95

턴 2 (플레이어):
  공격 15 → 슬라임 HP 15 → 0
  → 승리!

실제 플레이:
- 방어 없이도 쉽게 클리어
- 3-4턴이면 처치 가능
- HP 손실 10-15 정도
```

#### 디자인 의도
```
목적:
✅ 기본 전투 메커니즘 학습
✅ 블록 매치 → 공격 연결 이해
✅ 적 행동 예고 시스템 소개
✅ 자신감 부여

패턴 특징:
- 단순함 (2가지 행동만)
- 낮은 위협
- 예측 가능
- 실수해도 회복 가능
```

### 2. 고블린 (중급 - Normal Enemy)

#### 기본 정보
```
이름: 고블린
역할: 일반 적
난이도: ⭐⭐⭐☆☆
출현: 4-6번째 전투
```

#### 스탯
```
HP: 50
공격력: 8
방어력: 0
특수능력: 강공격
```

#### 행동 패턴
```
1. 공격 (60%)
   데미지: 8
   "고블린이 칼을 휘두른다!"

2. 강공격 (40%)
   데미지: 12
   예고: "고블린이 큰 칼을 휘두를 준비를 한다!"
   실행: "고블린의 강력한 일격!"

   행동 순서:
   턴 N: 강공격 선택 → 예고 표시
   턴 N+1: 강공격 실행 (12 데미지)
```

#### 밸런싱
```
목표 전투 시간: 5-8턴
예상 시나리오:

시나리오 A (공격 위주):
턴 1 (P): 공격 20 → 고블린 50 → 30
턴 1 (E): 공격 8 → 플레이어 100 → 92
턴 2 (P): 공격 20 → 고블린 30 → 10
턴 2 (E): 강공격 예고
턴 3 (P): 공격 15 → 고블린 10 → 0 (승리!)

시나리오 B (강공격 대비):
턴 1 (P): 공격 15 → 고블린 50 → 35
턴 1 (E): 강공격 예고 (12)
턴 2 (P): 방어 20 쌓기
턴 2 (E): 강공격 12 → 방어 20 → 8 (HP 무손상)
턴 3 (P): 공격 20 → 고블린 35 → 15
턴 3 (E): 공격 8 → 방어 8 → 0, HP 손상 없음
턴 4 (P): 공격 20 → 고블린 15 → 0 (승리!)

실제 플레이:
- HP 손실 15-25
- 방어 활용 시작
- 강공격 타이밍 학습
```

#### 디자인 의도
```
목적:
✅ 강공격 예고 시스템 학습
✅ 방어의 중요성 이해
✅ 타이밍 판단 연습
✅ 난이도 상승 체감

패턴 특징:
- 강공격으로 위협 증가
- 예고로 대응 가능
- 방어 쌓기 유도
- 전략적 선택 필요

실패 시나리오:
- 강공격 무시 → HP 큰 손실
- 방어 부족 → HP 30-40 손실
→ 다음 전투 위험
```

### 3. 오크 (보스 - Boss Enemy)

#### 기본 정보
```
이름: 오크
역할: 보스
난이도: ⭐⭐⭐⭐★
출현: 5번째 전투 (MVP 마지막)
```

#### 스탯
```
HP: 150
공격력: 15 (기본) / 20 (분노)
방어력: 0 (방어 행동 시 +10)
특수능력: 분노 (HP 50% 이하)
```

#### 행동 패턴
```
1. 공격 (50%)
   데미지: 15 (기본) / 20 (분노)
   "오크가 도끼를 내리친다!"

2. 강공격 (30%)
   데미지: 25 (기본) / 30 (분노)
   예고: "오크가 분노하며 도끼를 들어올린다!"
   실행: "오크의 강력한 도끼 일격!"

3. 방어 (20%)
   방어력: +10 (1턴)
   "오크가 방패를 들어올린다!"
```

#### 특수 능력: 분노
```
발동 조건:
HP 75 이하 (50% 이하)

효과:
- 공격력 +5 (영구)
- 공격: 15 → 20
- 강공격: 25 → 30

시각적 피드백:
"오크가 분노하며 붉게 물든다!"
오크 스프라이트: 빨간 오라 추가

전략적 의미:
- 후반 위협 급상승
- 빠른 처치 vs 안전한 방어
- 긴장감 유지
```

#### 밸런싱
```
목표 전투 시간: 10-15턴
예상 시나리오:

전반전 (HP 150 → 75):
턴 1 (P): 공격 25 → 오크 150 → 125
턴 1 (E): 공격 15 → 플레이어 100 → 85
턴 2 (P): 방어 20 쌓기
턴 2 (E): 강공격 예고 (25)
턴 3 (P): 공격 20 → 오크 125 → 105
턴 3 (E): 강공격 25 → 방어 20 → 5, HP 85 → 80
턴 4 (P): 공격 25 → 오크 105 → 80
턴 4 (E): 방어 +10
턴 5 (P): 공격 25 → 방어 10 → 실제 15, 오크 80 → 65
턴 5 (E): 공격 15 → 플레이어 80 → 65

후반전 (HP 75 이하, 분노):
턴 6 (P): 공격 30 → 오크 65 → 35
턴 6 (E): [분노 발동!] 강공격 예고 (30)
턴 7 (P): 방어 25 쌓기 (긴급)
턴 7 (E): 강공격 30 → 방어 25 → 5, HP 65 → 60
턴 8 (P): 공격 35 → 오크 35 → 0 (승리!)

최종 결과:
- 총 8턴 소요
- HP 손실: 40
- 방어 활용: 2회

실제 플레이 범위:
- 숙련: 8-10턴, HP 손실 30-40
- 평균: 10-13턴, HP 손실 40-60
- 미숙: 13-15턴, HP 손실 60-80
```

#### 디자인 의도
```
목적:
✅ MVP의 최종 도전
✅ 모든 메커니즘 활용
✅ 페이즈 전환 경험
✅ 성취감 제공

패턴 특징:
- 3가지 행동 (복잡도 증가)
- 높은 HP (긴 전투)
- 분노 페이즈 (긴장감)
- 방어 행동 (전략 다양화)

위기 포인트:
1. 첫 강공격 (턴 2-3)
   → 방어 부족 시 큰 손실
2. 분노 발동 (HP 50%)
   → 후반 위협 급증
3. 분노 중 강공격 (30)
   → 방어 필수

성공 조건:
- 블록 효율적 사용
- 방어 타이밍 정확
- HP/방어 균형 유지
- 분노 페이즈 대비
```

## 📊 밸런싱

### 전투 길이 목표

```
적별 턴 수:
┌─────────┬────────┬────────┬────────┐
│  적     │  최소  │  평균  │  최대  │
├─────────┼────────┼────────┼────────┤
│ 슬라임  │  3턴   │  4턴   │  5턴   │
│ 고블린  │  5턴   │  6턴   │  8턴   │
│ 오크    │  10턴  │  12턴  │  15턴  │
└─────────┴────────┴────────┴────────┘

전투 시간 (턴당 15-30초 가정):
- 슬라임: 1-2분
- 고블린: 2-4분
- 오크: 5-7분

이유:
- 너무 짧으면: 전략성 부족, 허무함
- 너무 길면: 지루함, 피로
- 적당한 길이: 긴장감 + 성취감
```

### 데미지 밸런스

```
플레이어 평균 공격 (3개 블록):
├─ 검: 15 (5×3)
├─ 도끼: 24 (8×3, 방어 -6)
└─ 화염: 9 + DOT 6 (3×3 + 2×3턴)

적 평균 데미지:
├─ 슬라임: 5 (기본)
├─ 고블린: 8 (기본) / 12 (강공격)
└─ 오크: 15 (기본) / 25 (강공격) / 20-30 (분노)

밸런스 원칙:
✅ 플레이어가 공격적으로 유리
   → 적 처치 속도 > 적 공격 속도
✅ 하지만 방어 관리 필수
   → 방어 없으면 HP 빠르게 소진
✅ 위험-보상 균형
   → 공격 집중 = 빠른 승리 but 위험
   → 방어 중심 = 안전 but 느림
```

### HP 밸런스

```
플레이어 HP: 100 (기본)

적별 예상 HP 손실:
├─ 슬라임: 10-15 (1-2회 피격)
├─ 고블린: 15-25 (2-3회 피격)
└─ 오크: 40-60 (5-7회 피격)

누적 손실 (3전투 연속):
├─ 슬라임 3회: 30-45
├─ 슬라임 + 고블린 2회: 40-65
└─ 슬라임 2 + 고블린 + 오크: 55-85

회복 없이 버틸 수 있는 전투:
- 완벽 플레이: 5-6전투
- 평균 플레이: 3-4전투
- 실수 많으면: 2전투

→ 회복/방어 관리 필수
```

### 방어력 밸런스

```
최대 방어력: 30 (기본)
방패 블록 3개: +15

적 공격력 vs 방어력:
├─ 슬라임 (5): 방어 15로 3회 무효화
├─ 고블린 (8/12): 방어 20으로 2-3회 무효화
└─ 오크 (15-30): 방어 30으로 1-2회 무효화

전략적 의미:
- 방어 = 2-3턴 안전
- 무적이 아님 (최대치 제한)
- 지속적 관리 필요
- 공격 vs 방어 선택

방어 중심 빌드 가능성:
- 방패 블록 다수 획득
- 방어 최대치 증가 유물
- 방어 중심 전략 플레이
→ 느리지만 안전한 플레이
```

### 난이도 곡선

```

   - Ascension 2: 적 HP +20%, 공격 +10%
   - ...
   - Ascension 20: 적 HP +200%, 공격 +100%

   추가 효과:
   - 적 행동 속도 증가
   - 엘리트 출현 빈도 증가
   - 보스 능력 강화
```
