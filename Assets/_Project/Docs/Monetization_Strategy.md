## 🎯 수익화 모델 개요

### 핵심 원칙

```
1. 재미 우선: 수익화가 게임플레이를 해치지 않음
2. 선택의 자유: 무료로도 완전한 경험 제공
3. 공정성: Pay-to-Win 없음
4. 투명성: 가격과 혜택 명확히 표시
```

### 선택된 모델: 하이브리드

```
기본 구조:
- PC: Premium 모델 ($4.99)
- Mobile: F2P + 광고 + 프리미엄 ($4.99)

플랫폼별 차별화:
- PC: 광고 없음, 순수 프리미엄
- Mobile: 광고 기반 무료 + 프리미엄 옵션
```

## 📱 광고 시스템 설계

### 1. 즉시 닫기 광고 (Interstitial Ads)

#### 표시 조건
```
기본 규칙:
- 마지막 광고 후 5분 경과 시

예외 조건 (광고 스킵):
❌ 첫 3회 사망 (신규 유저 보호)
❌ 1분 이내 재사망 (연속 실패)
❌ 보스전 중 사망 (몰입 보호)
❌ 프리미엄 유저
```

#### 플레이어 경험
```
30분 플레이 시나리오:
- 총 사망: 10회
- 광고 노출: 4-5회
- 총 방해 시간: 10-15초 (전체의 0.8%)

흐름:
사망 → [광고 1초] → X 클릭 → 즉시 게임 복귀
```

#### 수익
```
eCPM: $1.00
노출당 수익: $0.001-0.002
세션당 수익: $0.005-0.010
```

---

### 2. 보상형 광고 (Rewarded Ads)

#### A. 부활 시스템 (핵심 기능)

```
사망 시 선택지:
┌──────────────────────────┐
│   패배했습니다!           │
│                          │
│   [새로운 런 시작]        │  ← 기본
│                          │
│   🎬 [광고 보고 부활]    │  ← 선택
│   HP 50% 회복            │
│   방어 15 획득           │
│   (이번 런 1회 한정)     │
└──────────────────────────┘

제약:
- 런당 1회만
- 보스전에서만 제공 (전략적 가치)
- 첫 2전투에서는 불가
```

#### B. 보상 강화

```
전투 승리 후:
┌──────────────────────────┐
│  보상 선택:               │
│                          │
│  1️⃣ 도끼 블록 추가        │
│  2️⃣ 화염 블록 추가        │
│  3️⃣ 회복 +30             │
│                          │
│  🎬 광고 보고 2개 선택   │
│  (오늘 2/3회 남음)       │
└──────────────────────────┘

제약:
- 하루 3회까지
- 프리미엄: 무제한 + 광고 없음
```

#### C. 시작 보너스

```
런 시작 시:
┌──────────────────────────┐
│  새로운 모험 시작!        │
│                          │
│  [일반 시작]              │
│  HP: 100                 │
│                          │
│  🎬 [광고 보고 강화 시작]│
│  HP: 120                 │
│  방어: 10                │
│  (하루 3회)              │
└──────────────────────────┘
```

#### 수익
```
영상 광고 (30초):
eCPM: $3.00
노출당 수익: $0.02-0.03
세션당 수익: $0.04-0.06 (2회 시청)
```

### 3. 광고 빈도 최적화

```
30분 플레이 세션:

무료 플레이어:
├─ 즉시 닫기: 4-5회 (각 1-2초) = 10초
├─ 보상형 선택: 1-2회 (각 30초) = 60초
└─ 총 광고 시간: 70초 (전체의 3.9%)

느낌:
✅ "광고 좀 나오긴 하는데 참을 만함"
✅ "부활은 진짜 도움됨"
✅ "$5면 광고 없애는 것도 나쁘지 않네"
```

## 💎 프리미엄 모델

### 프리미엄 패키지 ($4.99)

```
포함 내용:
✅ 모든 광고 영구 제거
✅ 부활 자동 제공 (런당 1회, 광고 없이)
✅ 보상 2배 자동 (하루 3회, 광고 없이)
✅ VIP 아이콘/배지
✅ 향후 DLC 10% 할인 (계획)

가치 계산:
- 광고 시간 절약: 월 ~2시간
- 부활 편의성: 월 ~30회
- 보상 효율: 월 ~90회
→ 시간당 가치: $2.50
```

### 구매 동선

```
1차 접점: 첫 광고 노출 후
┌────────────────────────────┐
│  💎 광고 없이 즐기기?      │
│  [프리미엄 $4.99]          │
│  • 모든 광고 제거          │
│  • 부활 자동 제공          │
└────────────────────────────┘

2차 접점: 5회 광고 시청 후
"광고를 5회 시청하셨네요!"
"프리미엄으로 광고 제거하시겠어요?"

3차 접점: 부활 실패 후
"광고 없이 항상 부활하려면?"
"프리미엄 구매하기"
```

### 전환율 목표

```
목표 전환율: 10% (MAU 기준)
평균 전환 시점: 5-10시간 플레이 후
전환 이유:
- 광고 피로도 (60%)
- 부활 편의성 (25%)
- 게임 지지 (15%)
```

## 🚨 리스크 및 고려사항

### 1. 광고 피로도
```
문제:
- 광고 너무 많으면 → 부정적 리뷰
- 광고 너무 적으면 → 수익 부족

해결:
✅ 5분마다 제한
✅ 신규 유저 보호
✅ 보스전 보호
✅ 프리미엄 옵션 제공
```

### 2. 밸런싱
```
문제:
- 광고 보상이 너무 강하면 → 필수 요소화
- 광고 보상이 너무 약하면 → 무의미

해결:
✅ 광고 = 10% 도움 (편의성)
✅ 광고 없이도 클리어 가능
✅ A/B 테스트로 최적화
```

### 3. 플랫폼 정책
```
Steam:
❌ 광고 부적합 → PC는 프리미엄만

iOS:
✅ Unity Ads 지원
⚠️ 30% 수수료 (IAP)

Android:
✅ Google Ads 지원
⚠️ 30% 수수료 (IAP)
```
