## 🟢 초록 블록 (와일드카드)

> 현재 경로 색상과 연결 가능한 특수 블록

### WILDCARD - 별

```
ID: WILDCARD
이름: 별 (와일드카드)
색상: 초록 (Green)
아이콘: ⭐

효과: 연결된 블록 색상의 효과 적용

초기 풀: O (5%)
획득 시기: 초기 블록

특징:
- 현재 경로 색상 또는 다른 와일드카드와만 연결 가능
- 연결된 색상을 "따라감"
- 매우 희귀하고 강력

연결 규칙:
1. 경로 색상이 정해진 경우: 해당 색상 또는 와일드카드와만 연결
2. 와일드카드로 시작한 경우: 다음 블록이 경로 색상 결정
3. 결정된 경로 색상으로만 이어짐

예시 1 (일반):
🔴⚔️ 검 → 🟢⭐ 별 → 🔴🪓 도끼
→ 경로 색상 = 빨강
→ 별은 빨간색으로 취급됨
→ 모든 효과 발동

예시 2 (와일드카드 시작):
🟢⭐ 별 → 🟢⭐ 별 → 🔴⚔️ 검 → 🔴🪓 도끼
→ 검이 경로 색상을 빨강으로 결정
→ 이후 빨강 또는 별과만 연결 가능

밸런싱:
- 생성 확률: 5% (매우 희귀)
- 효과: 연결된 블록 효과
- 개수: 보드에 2-3개 이하

전략적 가치:
✅ 경로 연장에 도움
✅ 와일드카드 체인 가능
✅ 희귀하므로 신중히 사용

색상 전환 제약:
⚠️ 한 경로에서 한 색상만
⚠️ 와일드카드로 색상 "전환" 불가
⚠️ 예: 빨강 → 와일드 → 파랑 (X)
⚠️ 현재 경로 색상과만 연결: 빨강 → 와일드 → 빨강 (O)
```

  - 🛡️ 방패: 80%

### 블록 추가 시 비율 조정

```
예시 1: 도끼 추가
초기:
🔴 빨강: 25% (검 100%)

추가 후:
🔴 빨강: 25%
  - 검: 60%
  - 도끼: 20%
  - 나머지: 20%

→ 빨강 총 비율 유지

예시 2: 포션 추가 (중복)
초기:
🟤 갈색: 30%
  - 포션: 15%

추가 후:
🟤 갈색: 30%
  - 포션: 20%
  - 쓰레기: 75%
  - 버프: 5%

→ 포션 확률 증가
```

## 🎯 획득 우선순위

### MVP 블록 추가 순서

```
초기 (게임 시작):
1. 🔴⚔️ 검
2. 🔵🛡️ 방패
3. 🟡💰 금화
4. 🟤💩 쓰레기
5. 🟤🧪 포션
6. 🟤✨ 버프
7. 🟢⭐ 별

전투 1 보상:
- 🔴🪓 도끼 (공격 특화)
- 🔴🔥 화염 (DOT 특화)

전투 2-3 보상:
- 🟡💎 보석 (골드 특화)

→ 점진적 복잡도 증가
→ 플레이어 선택권 존중
```

### 추천 빌드

```
1. 공격 특화 빌드
   - 도끼 다수 획득
   - 버프 블록 추가
   - 빠른 전투 종료

2. 방어 특화 빌드
   - 방패 다수 획득
   - 안전한 장기전

3. 균형 빌드
   - 공격/방어 균등
   - 포션 확보
   - 안정적 플레이

4. 골드 특화 빌드
   - 보석 다수 획득
   - 상점 활용
   - 유물 파밍
```
